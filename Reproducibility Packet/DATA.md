# Data

## Feasibility-spike data

**What it is.** All current packet outputs are self-generated MuJoCo simulation results for the mechanics-only feasibility gate: fault-signature summaries, convergence measurements, an independent beam-comparison summary, and figures.

**How to get it.** No download or account is required. Run Step 3 in [`README.md`](README.md); the script deterministically regenerates the result files under `results/feasibility_spike/`.

**License and commercial-use status.** No third-party dataset is used or redistributed. The project-owned script and configuration are MIT-licensed.
